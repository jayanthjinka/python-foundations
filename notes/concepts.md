# Variables
- A variable is a name used to store a value.
- Variables allow us to reuse values in a program.
- Data types decide what kind of data a variable can hold.
- Clear variable names make code easy to read and understand.

# Control Flow
- if and else help a program make decisions.
- Conditions are checked as true or false.
- elif is used when there are multiple conditions.
- Control flow decides what code runs in different situations.
